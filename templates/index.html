CPU Scheduling

It's a process that allows one process to use the CPU while the execution of another is on hold (in waiting state)
due to the unavailability of any resource (like I/O).It is an essential part of a multitasking operating system.
The goal of CPU scheduling is to optimize CPU usage and provide an efficient environment for processes to run.

FCFS

First-Come, First-Served (FCFS) is the simplest CPU scheduling algorithm. Processes are executed in the order they arrive in the ready queue, leading to a non-preemptive system.

SJF

Shortest Job First (SJF) schedules processes based on the shortest execution time. It can be preemptive or non-preemptive and aims to minimize average waiting time.

Round Robin

Round Robin (RR) is a CPU scheduling algorithm that assigns a fixed time slice to each process in a cyclic order. It is preemptive and ensures that all processes are treated equally, avoiding starvation.

Priority

The Priority scheduling algorithm executes processes based on priority. Higher-priority processes are executed first, while lower-priority processes wait.
